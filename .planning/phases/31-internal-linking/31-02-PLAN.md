# Plan 31-02: Related Links Component & Cross-Page Navigation

## Context

This plan creates cross-page navigation to improve internal linking for SEO. Related links help users discover content and help search engines understand page relationships.

### Current State
- SEO pages have no "Related Links" or "Explore More" sections
- No cross-linking between location hierarchy and category pages
- Location pages don't link to related treatments
- Category pages don't link to locations

### Goal
Create a `RelatedLinks` component and integrate related links sections into key SEO pages to improve internal linking and user navigation.

## Tasks

### Task 1: Create RelatedLinks Component
**Type:** auto
**Why:** Reusable component for consistent related links styling across pages

Create `src/components/ui/relatedLinks.tsx`:

```tsx
import Link from 'next/link'
import { ArrowRight } from '@phosphor-icons/react/dist/ssr'

export interface RelatedLink {
  label: string
  href: string
  description?: string
}

interface RelatedLinksProps {
  title: string
  links: RelatedLink[]
  className?: string
}

/**
 * RelatedLinks - Cross-navigation component for SEO internal linking
 * Renders a titled section with links to related pages
 */
export function RelatedLinks({ title, links, className = '' }: RelatedLinksProps) {
  if (links.length === 0) return null

  return (
    <section className={`${className}`}>
      <h2 className="text-lg font-semibold text-text-primary mb-4">{title}</h2>
      <div className="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 gap-3">
        {links.map((link) => (
          <Link
            key={link.href}
            href={link.href}
            className="group flex items-center justify-between p-4 bg-glass-bg backdrop-blur-lg border border-glass-border rounded-xl hover:border-brand-primary/50 transition-colors"
          >
            <div>
              <span className="text-text-primary font-medium group-hover:text-brand-primary transition-colors">
                {link.label}
              </span>
              {link.description && (
                <p className="text-sm text-text-tertiary mt-1">{link.description}</p>
              )}
            </div>
            <ArrowRight
              size={16}
              weight="bold"
              className="text-text-tertiary group-hover:text-brand-primary transition-colors flex-shrink-0 ml-2"
            />
          </Link>
        ))}
      </div>
    </section>
  )
}
```

**Verify:**
- [ ] File created at `src/components/ui/relatedLinks.tsx`
- [ ] Component accepts `title`, `links` array, and optional `className`
- [ ] Glassmorphic styling consistent with design system
- [ ] Returns null when links array is empty
